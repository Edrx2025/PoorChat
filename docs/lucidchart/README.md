## Uso en Lucidchart

1. Abre un documento de Lucidchart.
2. Selecciona la opción **Diagram as Code**.
3. Elige **Mermaid**.
4. Abre uno de los archivos `.mmd`, copia todo su contenido y pégalo en el
   editor.
5. Genera el diagrama y repite el proceso para las demás figuras.

Lucidchart crea el diagrama desde el código Mermaid. Las imágenes PNG se
incluyen únicamente para comparar el resultado con el Word.
