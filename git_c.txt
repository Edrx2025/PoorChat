/* Traer cambios
 * git fetch origin                     - traer cambios de las ramas remotas
 * git checkout remotes/origin/branch   - pruebo cambios y confirmo el pull request
 * 
 * git checkout main
 * git pull origin main                 - traes cambios de origin main (remoto) -> main (local)
 * 
 * git checkout branch                  - me muevo a mi rama
 * git merge main                       - mergeo cambios del main a mi rama
